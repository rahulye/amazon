# Amazon site (JavaScript)
A vanilla JavaScript e-commerce frontend built to practice **DOM manipulation, cart logic, checkout flow, and modular JavaScript architecture** without frameworks.

## Tech Stack
- HTML
- CSS
- JavaScript 

## Preview
[Live demo](https://amazon-esite.vercel.app/)

## Why?
- To improve DOM manipulation without frameworks
- To build cart, checkout, and order tracking logic from scratch
- To practice modular JavaScript architecture
